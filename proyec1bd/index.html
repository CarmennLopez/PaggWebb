Introducción

Esta es una página de ejemplo para demostrar la integración de CSS3 en un documento HTML. A medida que avancemos, aplicaremos diferentes estilos para mejorar la apariencia.

El objetivo principal es que te familiarices con cómo vincular hojas de estilo externas y cómo aplicar reglas CSS a elementos específicos de tu HTML.
